# Usage
<!-- usage -->
```sh-session
$ npm install -g arjancli
$ arjan COMMAND
running command...
$ arjan (-v|--version|version)
arjancli/0.0.0 linux-x64 node-v12.13.1
$ arjan --help [COMMAND]
USAGE
  $ arjan COMMAND
...
```
