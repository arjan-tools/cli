Arjan Localize

Arjan Localize es módulo de nodo para localizar y traducir automáticamente sitios html. También funciona como un sistema básico de administración de contenido para sitios estáticos

Pruébalo

La localización de html consiste en analizar el documento html, extraer identificaciones en elementos con texto, y crear una configuración regional JSON que contenga todos los ID del documento y el contenido de texto.

Utiliza el sistema de traducción automática neuronal de AWS que se utiliza en amazon.com.
